 # Bank Management System 
 
## Built a console-based Bank Management System using Core Java to replicate essential banking operations with security and data integrity features.

## Key Features:
1.Account creation with validation to prevent duplicate account numbers

2.PIN-based authentication for sensitive operations like deposits, withdrawals, and balance inquiries

3.Deposit and withdrawal capabilities with proper balance validation

4.Balance check and list of all account holders

5.Menu-driven interface for seamless user interaction

6.Structured using Object-Oriented Programming principles

## Technologies Used: Java, OOP Concepts, Arrays, Exception Handling, Scanner (for I/O)
